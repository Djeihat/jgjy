# Community Material Flow Map

A project by Jay Yamakawa and Justin Garofoli.

An interactive map visualizing the material inputs and waste outputs of
geographic areas — from large regions down to the neighborhood level. The
core civic hook: **"How much garbage is my community producing?"**
